# Implement Job Post Creation with EGP Currency

**Description:**  
Develop the "Post Job" screen for employers with salary fields in Egyptian Pounds (EGP) and Egyptian-specific job categories.

**Priority:**  
High

**Complexity:**  
Medium

**Notes:**  
- **Reference:** Use `@UI/job_post_creation_screen/code.html`.
- Backend: Create a `jobs` record in Supabase using the PostgREST API or Supabase Flutter Client.
- Form fields: Replace "SAR" with "ج.م" (EGP) as the currency unit.
