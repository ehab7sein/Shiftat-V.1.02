# Initialize Flutter Project and RTL Configuration

**Description:**  
Set up the Flutter project with a clean architecture (e.g., Bloc or Provider) and global RTL (Right-to-Left) support for the Arabic interface.

**Priority:**  
High

**Complexity:**  
Easy

**Notes:**  
- Configure `MaterialApp` with `localizationsDelegates` and `supportedLocales` set to 'ar'.
- Setup custom theme using colors from `@UI/splash_screen/code.html` (Primary: #1c74e9).
